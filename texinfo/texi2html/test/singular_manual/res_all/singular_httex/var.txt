5.1.133 var
-----------
Syntax:
var ( int_expression )
Type:
poly
Purpose:
var(n) returns the n-th ring variable.
Example:
  ring r=0,(x,y,z),dp;
  var(2);
==> y
See
int;
nvars;
ring;
varstr.
