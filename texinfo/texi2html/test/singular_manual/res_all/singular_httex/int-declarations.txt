4.3.1 int declarations
----------------------

Syntax:
int name = int_expression ;

Purpose:
defines an integer variable.

Default:
0

Example:
  int i = 42;
  int j = i + 3; j;
==> 45
  i = i * 3 - j; i;
==> 81
  int k;   // assigning the default value 0 to k
  k;
==> 0
