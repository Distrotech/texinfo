5.1.108 repart
--------------

Syntax:
repart ( number_expression )
Type:
number
Purpose:
returns the real part of a number from a complex ground field,

returns its argument otherwise.
Example:
  ring r=(complex,i),x,dp;
  repart(1+2*i);
==> 1
See
impart.
