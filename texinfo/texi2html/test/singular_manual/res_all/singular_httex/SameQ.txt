D.7.5.16 SameQ
..............
Procedure from library zeroset.lib (see zeroset_lib).

Usage:
SameQ(a, b); a,b=list/intvec

Purpose:
test a == b elementwise, i.e., a[i] = b[i].

Return:
int

0 if a != b

1 if a == b
