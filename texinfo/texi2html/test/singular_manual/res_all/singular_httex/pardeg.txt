5.1.93 pardeg
-------------
Syntax:
pardeg ( number_expression )
Type:
int
Purpose:
returns the degree of a number considered as a polynomial in the ring parameters.
Example:
  ring r=(0,a,b,c),(x,y,z),dp;
  pardeg(a^2*b);
==> 3
See
deg;
number;
ring;
var.
