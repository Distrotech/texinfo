5.1.84 npars
------------
Syntax:
npars ( ring_name )
Type:
int
Purpose:
returns the number of parameters of a ring.
Example:
  ring r=(23,t,v),(x,a(1..7)),lp;
  // the parameters are t,v
  npars(r);
==> 2
See
par;
parstr;
ring.
