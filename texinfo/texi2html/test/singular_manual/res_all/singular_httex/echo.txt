5.3.2 echo
----------
Type:
int
Purpose:
input is echoed if echo >= voice.

echo is a local setting for a procedure and defaulted to 0.

echo does not affect the output of commands.
Example:
echo = 1;
int i = echo;
==> int i = echo;
See
int;
voice.
