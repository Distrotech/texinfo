D.4.4.19 modulesEqual
.....................
Procedure from library mprimdec.lib (see mprimdec_lib).

Usage:
modulesEqual(N1, N2) N1, N2 modules,

Return:
int i

if (N1==N2) then i=1

else i=0

Example:
LIB "mprimdec.lib";
ring  r = 0,(x,y,z),dp;
module N1=x*freemodule(2);
module N2=xy*freemodule(2);
int i=modulesEqual(N1,N2);
i;
==> 0
N2=N1;
i=modulesEqual(N1,N2);
i;
==> 1
