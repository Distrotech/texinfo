D.3.2.10 det_B
..............
Procedure from library linalg.lib (see linalg_lib).

Usage:
det_B(A); A any matrix

Return:
returns the determinant of A

Note:
the computation uses the busadj algorithm

Example:
LIB "linalg.lib";
ring r=0,(x),dp;
matrix A[10][10]=random(2,10,10)+unitmat(10)*x;
print(A);
det_B(A);
