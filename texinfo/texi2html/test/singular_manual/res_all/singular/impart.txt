5.1.49 impart
-------------

Syntax:
impart ( number_expression )
Type:
number
Purpose:
returns the imaginary part of a number in a complex ground field,

returns 0 otherwise.
Example:
  ring r=(complex,i),x,dp;
  impart(1+2*i);
==> 2
See
repart.
