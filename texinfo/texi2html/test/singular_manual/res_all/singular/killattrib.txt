5.1.58 killattrib
-----------------
Syntax:
killattrib ( name, string_expression )
Type:
none
Purpose:
deletes the attribute given as the second argument.
Example:
  ring r=32003,(x,y),lp;
  ideal i=maxideal(1);
  attrib(i,"isSB",1);
  attrib(i);
==> attr:isSB, type int
  killattrib(i,"isSB");
  attrib(i);
==> no attributes
See
attrib;
option.
