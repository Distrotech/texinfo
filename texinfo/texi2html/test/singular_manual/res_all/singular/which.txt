D.2.2.17 which
..............
Procedure from library general.lib (see general_lib).

Usage:
which(command); command = string expression

Return:
Absolute pathname of command, if found in search path.
Empty string, otherwise.

Note:
Based on the Unix command 'which'.

Example:
LIB "general.lib";
which("sh");
==> /bin/sh
