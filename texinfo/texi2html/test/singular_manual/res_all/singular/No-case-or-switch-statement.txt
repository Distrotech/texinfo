6.2.3 No case or switch statement
---------------------------------

SINGULAR does not offer a case (or switch)
statement. However, it can be imitated in the following way:

while (1)
{
   if (choice == choice_1) { ...; break;}
   ...
   if (choice == choice_n) { ...; break;}
   // default case
   ...; break;
}
