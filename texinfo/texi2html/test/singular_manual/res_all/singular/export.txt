5.2.5 export
------------
Syntax:
export name ;

export list_of_names ;
Purpose:
converts a local variable of a procedure to a global one.
Note:
Objects defined in a ring are not automatically exported
when exporting the ring (use keepring instead).
Example:
proc p1
{
  int i,j;
  export(i);
  intmat m;
  listvar();
  export(m);
}
p1();
==> // m                    [1]  intmat 1 x 1
==> // j                    [1]  int 0
==> // i                    [0]  int 0
listvar();
==> // m                    [0]  intmat 1 x 1
==> // i                    [0]  int 0

See
keepring.
