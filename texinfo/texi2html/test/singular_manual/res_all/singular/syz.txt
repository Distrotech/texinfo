5.1.126 syz
-----------
Syntax:
syz ( ideal_expression )

syz ( module_expression )
Type:
module
Purpose:
computes the first syzygy (i.e., the module of relations of the given
generators) of the ideal, resp. module.
Example:
  ring R=0,(x,y),(c,dp);
  ideal i=x,y;
  syz(i);
==> _[1]=[y,-x]
