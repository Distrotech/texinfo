5.1.117 sortvec
---------------
Syntax:
sortvec ( ideal_expression )

sortvec ( module_expression )
Type:
intvec
Purpose:
computes the permutation {\tt v}
which orders the ideal, resp.\ module, {\tt I} by its initial terms,
starting with the smallest, that is, {\tt I(v[i]) < I(v[i+1])} for all
{\tt i}.
Example:
  ring r=0,(x,y,z),dp;
  ideal I=y,z,x,x3,xz;
  sortvec(I);
==> 2,1,3,5,4
See
general_lib.
