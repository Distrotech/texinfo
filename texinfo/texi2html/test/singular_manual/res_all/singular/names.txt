5.1.82 names
------------
Syntax:
names ( )

names ( ring_name )
Type:
list of strings
Purpose:
returns the names of all user-defined variables which are ring independent
(this includes the names of  procedures) or, in the second case, which
belong to the given ring.

Example:
  int i=9;
  ring r;
  poly f;
  poly g;
  setring r;
  names();
