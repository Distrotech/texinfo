5.1.8 close
-----------
Syntax:
close ( link_expression )
Type:
none
Purpose:
closes a link.
Example:
link l="MPtcp:launch";
open(l); // start SINGULAR "server" on localhost in batchmode
close(l); // shut down SINGULAR server
See
link;
open.
