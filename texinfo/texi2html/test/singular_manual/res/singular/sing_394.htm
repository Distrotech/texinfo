D.2.2.2 ASCII

Procedure from library general.lib (see general_lib).

Usage:

ASCII([n,m]); n,m= integers (32 <= n <= m <= 126)

Return:

string of printable ASCII characters (no native language support) ASCII(): string of all ASCII characters with its numbers, ASCII(n): n-th ASCII character
ASCII(n,m): n-th up to m-th ASCII character (inclusive)

Example: 

LIB "general.lib";
ASCII();"";
