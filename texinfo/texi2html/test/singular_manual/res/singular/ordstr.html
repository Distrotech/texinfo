5.1.91 ordstr

Syntax:

ordstr ( ring_name )

Type:

string

Purpose:

returns the description of the monomial ordering of the ring.

Example:
  ring r=7,(x,y),wp(2,3);
  ordstr(r);
→ wp(2,3),C

See charstr; parstr; ring; varstr.
